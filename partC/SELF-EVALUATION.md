# Self-Evaluation

## Could I write this code myself without AI today?

Partially. I can explain the project structure, the React UI state, the Express route flow, and the loan rule functions. I would need more time to write all documents, OpenAPI content, and tests manually, but the main architecture is understandable.

## What would I do differently next time?

I would create the GitHub repository before starting local commits, then push after each phase immediately. I would also install dependencies earlier and test the frontend in a browser as soon as the first screen exists.

## What did I learn?

I learned that AI is most useful when the task is split into planning, implementation, and reflection. I also learned that Git environment verification is important because local folders may belong to a different parent repository. The best result came from using AI for drafts while still checking commands, files, security risks, and tests myself.
