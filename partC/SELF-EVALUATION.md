# Өөрийн үнэлгээ

## Би энэ кодыг өнөөдөр AI-гүйгээр өөрөө бичиж чадах уу?

Хэсэгчлэн чадна. Би төслийн бүтэц, React UI state, Express route flow, loan rule функцүүдийг тайлбарлаж чадна. Бүх document, OpenAPI агуулга, тестүүдийг гараар бичихэд илүү их хугацаа хэрэгтэй боловч үндсэн архитектур ойлгомжтой.

## Дараагийн удаа юуг өөрөөр хийх вэ?

Local commit эхлэхээс өмнө GitHub repository үүсгээд, phase бүрийн дараа шууд push хийнэ. Мөн dependency-үүдийг эрт суулгаж, эхний screen гармагц frontend-ийг browser дээр шалгана.

## Би юу сурсан бэ?

AI нь task-ийг planning, implementation, reflection гэж хуваах үед хамгийн үр дүнтэй гэдгийг сурсан. Мөн Git environment verification чухал гэдгийг ойлгосон. Учир нь local folder өөр parent repository-д хамаарч байж болно. Хамгийн сайн үр дүн нь AI-г draft хийхэд ашиглаад, command, file, security risk, test-үүдийг өөрөө шалгаснаас гарсан.
